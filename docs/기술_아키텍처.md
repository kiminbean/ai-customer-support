                              │ - 차원: 3072      │
                              └──────────────────┘

#### 검색 전략

| 검색 방식 | 설명 | 용도 |
|-----------|------|------|
| **의미 검색** | 벡터 유사도 (코사인) | 자연어 질문, 유사 의미 매칭 |
| **키워드 검색** | BM25 | 정확한 용어, 주문번호 등 |
| **하이브리드** | 의미 + 키워드 결합 (RRF) | 기본 검색 전략 |
| **리랭킹** | Cross-encoder로 결과 재순위 | 정확도 최적화 |

```python
# 하이브리드 검색 예시 (Pseudo-code)
def hybrid_search(query: str, top_k: int = 5):
    # 1. 의미 검색
    semantic_results = vector_db.search(
        embedding=embed(query),
        top_k=top_k * 2,
        filter={"tenant_id": tenant_id}
    )
    
    # 2. 키워드 검색
    keyword_results = bm25_index.search(
        query=query,
        top_k=top_k * 2,
        filter={"tenant_id": tenant_id}
    )
    
    # 3. Reciprocal Rank Fusion
    fused = reciprocal_rank_fusion(
        semantic_results, keyword_results,
        weights=[0.6, 0.4]
    )
    
    # 4. Cross-encoder 리랭킹
    reranked = cross_encoder.rerank(query, fused[:top_k * 2])
    
    return reranked[:top_k]
```

### 5.3 PII (개인식별정보) 처리

```python
# PII 마스킹 파이프라인
class PIIMasker:
    patterns = {
        'phone': r'\d{2,3}-\d{3,4}-\d{4}',
        'email': r'[\w.-]+@[\w.-]+\.\w+',
        'rrn': r'\d{6}-[1-4]\d{6}',  # 주민번호
        'card': r'\d{4}[-\s]?\d{4}[-\s]?\d{4}[-\s]?\d{4}',
        'account': r'\d{3,4}-\d{2,6}-\d{4,6}',  # 계좌번호
    }
    
    def mask(self, text: str) -> tuple[str, dict]:
        """
        PII를 마스킹하고, 원본 매핑을 안전하게 저장
        LLM에는 마스킹된 텍스트만 전달
        """
        masked_text = text
        pii_map = {}
        for pii_type, pattern in self.patterns.items():
            for match in re.finditer(pattern, text):
                token = f"[{pii_type.upper()}_{len(pii_map)}]"
                pii_map[token] = match.group()
                masked_text = masked_text.replace(match.group(), token)
        return masked_text, pii_map
```
