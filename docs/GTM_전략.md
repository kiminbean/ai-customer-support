   → 인앱 팝업: "잠깐! 다운그레이드 옵션을 확인해 보세요"
   → 할인 오퍼: "3개월간 30% 할인 유지"

⚠️ 해지 요청
   → 해지 사유 설문 (필수)
   → 사유별 맞춤 제안:
     "비싸요" → 할인 or Free 다운그레이드
     "기능 부족" → 로드맵 공유 + 피드백 반영 약속
     "잘 안 돼요" → 1:1 지원 세션 제공
   → 해지 후: 30/60/90일 윈백 이메일 시퀀스
```
