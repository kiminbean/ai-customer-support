## 2. 타겟 시장 & 세분화

### 2.1 Primary Target Segments (우선순위)

#### 세그먼트 1: 이커머스 (최우선)

**특성:**
- 고객 문의 볼륨: 일 평균 500–10,000건
- 반복 문의 비율: 60–70%
- 야간/주말 응대 필요성: 매우 높음
- 다국어 지원 필요성: 역직구 고객 증가

**기업 유형:**
- 플랫폼형 이커머스 (쇼핑몰, 마켓플레이스)
- 브랜드 직영 쇼핑몰
- 오더 매칭 플랫폼

**접근 전략:**
- 직접 세일즈 (B2B SaaS 특성)
- 이커머스 컨퍼런스 참여
- 마케팅 파트너십
- SaaS 마켓플레이스 (쇼핑몰 빌더 연동)

**성공 지표:**
- 10개 파일럿 고객 (이커머스)
- 월 대화량 10,000건 이상
- AI 자동 해결률 70%+

#### 세그먼트 2: 핀테크 (고가치)

**특성:**
- 규제 요구사항 높음 (금융감독원 규정)
- 보안 민감도 최상
- 고객당 가치 높음 → 높은 지불 의향
- 24/7 긴급 대응 필요

**기업 유형:**
- 디지털 뱅킹 (카카오뱅크, 토스뱅크)
- 핀테크 대출 플랫폼
- 간편 결제 서비스
- 투자 플랫폼

**접근 전략:**
- 대면 미팅 (C-Level 접근)
- 정부/금융 단체 네트워크 활용
- 보안 인증(ISO 27001) 강조
- 금융감독원 규정 준수 가이드 제공

**성공 지표:**
- 5개 파일럿 고객 (핀테크)
- 계약 금액 5,000만원 이상
- SLA 99.9% 달성

#### 세그먼트 3: SaaS/IT (기술 친화적)

**특성:**
- 기술 친화적, AI 도입 장벽 낮음
- 영어/한국어 지원 필요
- 개발자/기술 사용자 대상 지원
- 티켓 기반 지원 체계

**기업 유형:**
- B2B SaaS (CRM, HR, ERP 등)
- 개발자 도구
- 데이터 분석 플랫폼

**접근 전략:**
- 개발자 마케팅 (DevRel)
- 오픈 소스 프로젝트 연계
- 커뮤니티 참여
- API 문서 중심 접근

**성공 지표:**
- 8개 파일럿 고객 (SaaS)
- API 연동 성공률 90%+
- 개발자 참여도 높음

### 2.2 Secondary Target Segments

#### 세그먼트 4: 통신

- SKT, KT, LGU+
- 통신 요금제/부가 서비스 고객센터
- 접근: SI 업체 파트너십

#### 세그먼트 5: 여행/항공

### 2.3 Non-Target Segments (제외)

## 4. 가격 전략

### 4.1 가격 전략 유형

**전략:** 혼합 가격 전략 (Mixed Pricing)

| 구분 | 전략 | 적용 |
|------|------|------|
| **구독료** | 수량 기반 가격 | Starter/Growth/Enterprise |
| **설정비** | 브레이크아웃 가격 | 초기 도메인 학습, 연동 |
| **사용량** | 슬라이딩 스케일 | 초과 대화 건당 과금 |
| **프리미엄** | 가치 기반 가격 | 고급 기능, 커스텀 솔루션 |

### 4.2 가격 접근성 분석

| 플랜 | 월 요금 | 연간 할인 | 접근성 |
|------|---------|-----------|--------|
| Starter | 49만원 | 39만원 (20%) | 중소기업 접근 가능 |
| Growth | 149만원 | 119만원 (20%) | 성장 중 기업 필수 |
| Enterprise | 499만원+ | 협의 | 대기업 필수 |

**가격 대비 가치 비교:**
- Starter: 49만원/월 → 5,000건 대화 = 건당 98원 (저렴)
- Growth: 149만원/월 → 25,000건 대화 = 건당 6원 (매우 저렴)
- Enterprise: 499만원+/월 → 무제한 = 건당 0원 (사용자 입장에서 무료)

### 4.3 파격 가격 전략 (초기)

**Phase 1 (M1-M3):**
- Starter: 39만원/월 (할인 20%)
- Growth: 119만원/월 (할인 20%)
- 초기 10사 무료 체험 (POC)

**Phase 2 (M6):**
- 정상 가격으로 전환
- 파일럿 고객 추가 (50사)

**Phase 3 (M12):**
- Enterprise 가격 단순화
- 대규모 계약 시 맞춤 가격

### 4.4 가격 전환 전략

```
Starter → Growth → Enterprise

가격 전환 유도:
1. 사용량 한도 초과 시 Growth 플랜 추천
2. 고급 분석/다국어 필요 시 Growth → Enterprise
3. 엔터프라이즈 보안(SSO, On-premise) 필요 시 Enterprise
```

## 6. 고객 획득 프로세스 (Sales Funnel)

### 6.1 Funnels 구조

```
무료 체험 → POC → 계약 → 온보딩 → 활성화 → 리텐션 → 확장
   ↓           ↓         ↓         ↓         ↓         ↓        ↓
  1,000명    100명     50명      40명      35명      33명     30명
(월간)      (POC)     (계약)    (온보딩)  (활성화)  (리텐션) (확장)
   ↓           ↓         ↓         ↓         ↓         ↓        ↓
    └─────────────────────────────────────────────────────────────┘
                              CAC: 450만원
```

### 6.2 단계별 프로세스

#### 단계 1: 무료 체험 (Free Trial)

**대상:** Starter/Growth 고객

**기간:** 14일

**제공:**
- 웹 위젯 무료
- 5,000건/월 대화
- 기본 분석 대시보드
- 기술 지원

**활성화 전략:**
- 1:1 온보딩 지원
- Quick Start 가이드
- 사용자 매뉴얼
- 고객 성공 매니저 연계

**성공 지표:**
- 체험자 활성화율: 30%+
- 체험 중 대화량: 2,000건 이상
- 체험 후 계전률: 40%+

#### 단계 2: POC (Proof of Concept)

**대상:** Enterprise, 중견기업

**기간:** 30–60일

**제공:**
- 맞춤 AI 모델 학습
- 전체 채널 연동
- 고급 분석
- 실제 운영 환경

**평가 기준:**
- AI 정확도: 85%+
- CSAT: 4.0/5.0+
- 비용 절감: 40% 이상
- 운영 효율: 50% 향상

**성공 지표:**
- POC 성공률: 60%+
- POC 기간: 평균 45일
- POC 후 계전률: 80%+

#### 단계 3: 계약 (Deal Closing)

**대상:** POC 성공 고객

**절차:**
- 가격 협상
- 계약 체결
- 결제 (계좌이체/계약서)

**가격 전략:**
- 파일럿 고객: 20% 할인
- 대규모 계약: 협의 가격
- 장기 계약 (3년): 25% 할인

#### 단계 4: 온보딩 (Onboarding)

**대상:** 신규 계약 고객

**기간:** 2–4주

**절차:**
1. 계약 승인 → 고객사 프로젝트 팀 구성
2. 도메인 데이터 수집 (FAQ, 매뉴얼 등)
3. AI 모델 학습 (2–3일)
4. 시스템 연동 (1–2일)
5. 교육 및 테스트
6. 라이브 배포

**성공 지표:**
- 온보딩 완료율: 90%+
- 온보딩 기간: 평균 21일
- 첫 7일 활성도: 80%+

#### 단계 5: 활성화 (Activation)

**대상:** 온보딩 완료 고객

**목표:**
- 첫 30일 내 1,000건 이상 대화
- AI 자동 해결률 60%+
- 고객 만족도 4.0/5.0+

**활성화 프로그램:**
- 첫 30일 가이드
- 질문/답변 템플릿 제공
- 고객 성공 매니저 주간 리뷰
- 피드백 루프 구축

#### 단계 6: 리텐션 (Retention)

**대상:** 활성 고객

**목표:**
- 월간 이탈률 < 2%
- NRR > 120%

**리텐션 전략:**
- 정기 케어 미팅 (월 1회)
- 성과 보고서 (분기 1회)
- 기능 업데이트 소식
- 고객 포털 활용성 유도

#### 단계 7: 확장 (Expansion)

**대상:** 활성 고객

**목표:**
- 플랜 업그레이드 (Starter → Growth → Enterprise)
- 추가 채널 연결
- 프리미엄 애드온 구매

**확장 전략:**
- 사용량 증가 시 Growth 플랜 제안
- 고급 기능 필요 시 Enterprise 제안
- 다국어 필요 시 프리미엄 애드온
