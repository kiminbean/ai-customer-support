## 1. 경쟁 구도 개요

AI 고객지원 시장은 크게 **글로벌 대형 SaaS**, **AI 네이티브 스타트업**, **국내 플레이어**로 구분됩니다.

```
                    기능 풍부
                       ▲
                       │
      Zendesk AI ●     │     ● Intercom Fin
                       │
   Freshdesk ●         │        ● Ada
                       │
 ─────────────────────●────────────────────► AI 고도화
       가격 높음        │  Our Position    가격 합리적
                       │     ★
    네이버 클로바 CX ●  │
                       │     ● Tidio
   카카오 i 커넥트 ●   │
                       │
                       │
                    기능 기본
```

## 5. 우리의 차별화 포인트

### 5.1 핵심 차별화 요소

#### 1️⃣ Deep Agents (멀티에이전트 시스템)

```
고객 문의 → [Routing Agent] → 유형 분류
                ↓
         ┌──────┴──────┐
         ↓             ↓
   [Answer Agent]  [Escalation Agent]
    RAG 기반 답변    복잡 문의 → 인간 전달
         ↓
   [Feedback Agent]
    품질 모니터링 & 학습
```

- **경쟁사 대부분**: 단일 AI 봇 구조
- **우리**: 역할 분담된 에이전트가 협업하여 복합 문의 처리
- **효과**: 해결률 30%+ 향상, 에스컬레이션 40% 감소

#### 2️⃣ RAG 기반 정확한 답변

- 기업 문서(FAQ, 매뉴얼, 정책)를 벡터 DB에 저장
- 질문마다 관련 문서를 실시간 검색 → 정확한 근거 기반 답변
- **출처 표시**: "이 답변은 [반품 정책 v2.3]을 기반으로 합니다"
- **환각 최소화**: 모르는 질문은 솔직히 "모르겠다" + 에스컬레이션

#### 3️⃣ 한국어 특화

| 기능 | 상세 |
|------|------|
| 존댓말/반말 자동 조절 | 고객 설정에 따른 톤 변환 |
| 한국식 CS 매너 | "안녕하세요~", "감사합니다 😊" 등 자연스러운 표현 |
| 한국어 형태소 분석 | 조사, 어미 변화 정확 처리 |
| 한국 비즈니스 맥락 | 배송사 연동, 한국 결제 시스템 이해 |
| 한글 오타 보정 | "교환해주세요" ↔ "교환해줴요" 등 |

#### 4️⃣ 합리적 가격

- Free 플랜: 스타트업/소규모 사업자 무료 사용
- Pro ₩99,000/월: 글로벌 경쟁사 대비 **50~70% 저렴**
- Enterprise: 네이버 클로바 대비 **80%+ 저렴**
- 투명한 가격: 숨겨진 비용 없음

## 6. SWOT 분석

### 6.1 Strengths (강점)

| # | 강점 | 상세 |
|---|------|------|
| S1 | **Deep Agents 기술** | 멀티에이전트 시스템으로 복합 문의 처리 능력 차별화 |
| S2 | **RAG 기반 정확성** | 환각 최소화, 출처 기반 신뢰도 높은 답변 |
| S3 | **한국어 네이티브** | 한국 시장에 최적화된 AI, 문화적 맥락 이해 |
| S4 | **합리적 가격** | Free 플랜 + 경쟁사 대비 50~80% 저렴 |
| S5 | **빠른 설정** | 5분 내 설치, 코드 한 줄 위젯 연동 |
| S6 | **최신 기술 스택** | 레거시 없이 최신 AI/클라우드 아키텍처 |

### 6.2 Weaknesses (약점)

| # | 약점 | 대응 전략 |
|---|------|-----------|
| W1 | **브랜드 인지도 부재** | PLG + 콘텐츠 마케팅으로 인지도 구축 |
| W2 | **레퍼런스 부족** | 무료 PoC → 초기 성공 사례 확보 |
| W3 | **기능 완성도** | MVP 집중 → 고객 피드백 기반 빠른 반복 |
| W4 | **팀 규모** | 핵심 인력 우선 확보, 점진적 확장 |
| W5 | **자금 한계** | 효율적 번 레이트 관리, 매출 기반 성장 |

### 6.3 Opportunities (기회)

| # | 기회 | 활용 전략 |
|---|------|-----------|
| O1 | **AI 시장 폭발적 성장** | 시장 성장 파도 위에서 빠른 스케일링 |
| O2 | **한국 중소기업 AI 도입 격차** | 저가 고품질 솔루션으로 미개척 시장 공략 |
| O3 | **정부 AI 지원 정책** | AI 바우처, R&D 지원금 적극 활용 |
