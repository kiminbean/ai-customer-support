### 2.4 Net Revenue Retention (NRR)

```
NRR = (기존 ARR + 확장 - 축소 - 이탈) / 기존 ARR

연간 계산:
- 기존 ARR: 100%
- 플랜 업그레이드: +15%
- 사용량 증가: +18%
- 추가 기능 구매: +5%
- 다운그레이드: -5%
- 이탈: -8%
────────────────────
NRR = 125%

→ 신규 고객 없이도 기존 매출이 연 25% 성장
```

## 4. 손익분기점 분석

### 4.1 BEP (Break-Even Point) 계산

```
월간 고정비: 약 2.1억원 (Y2 기준)
  - 인건비: 1.5억원
  - 사무실/운영: 0.2억원  
  - 마케팅 기본: 0.3억원
  - 기타: 0.1억원

건당 공헌이익: 
  - 평균 월 구독 수익/고객: 150만원
  - 변동비 (AI API + 인프라): -33만원
  - 공헌이익: 117만원/고객/월

BEP 고객 수 = 고정비 / 공헌이익
            = 2.1억원 / 117만원
            = 약 180개 고객

→ Y2 Q2 (누적 205고객) 시점에서 월간 흑자 전환
→ Y3 Q1 시점에서 누적 흑자 전환
```
