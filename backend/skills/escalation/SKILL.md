# Escalation Skill

## 이름
escalation — 상담원 에스컬레이션

## 설명
AI가 처리하기 어려운 상황을 감지하고 인간 상담원에게 전달합니다.

## 트리거 조건
1. **고객 요청**: "상담원 연결", "사람과 대화" 등 명시적 요청
2. **부정적 감정**: 분노, 불만, 항의 키워드 감지
3. **복잡한 문의**: 법적 문제, 피해보상, 개인정보 관련
4. **낮은 신뢰도**: AI 응답 신뢰도 < 0.3

## 입력
- `query` (string): 사용자 메시지
- `confidence` (float): 이전 응답의 신뢰도 점수

## 출력
```json
{
  "answer": "상담원 연결 안내 메시지...",
  "escalation": {
    "reason": "고객 요청",
    "priority": "보통",
    "conversation_id": "uuid"
  },
  "agent": "escalation_agent"
}
```

## 우선순위
| 사유 | 우선순위 |
|------|---------|
| 부정적 감정 감지 | 높음 |
| 복잡한 문의 | 높음 |
| 고객 요청 | 보통 |
| 낮은 응답 신뢰도 | 낮음 |

## 동작
1. 키워드 기반 감정/의도 분석
2. 에스컬레이션 사유 분류
3. 우선순위 결정
4. 안내 메시지 생성 (연락처, 운영시간 포함)

## 사용 에이전트
- `escalation_agent`: 직접 호출
- `orchestrator`: 의도 분류 시 자동 라우팅
- `faq_agent`: 낮은 신뢰도 시 자동 에스컬레이션
