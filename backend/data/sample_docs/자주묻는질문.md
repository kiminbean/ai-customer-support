# 자주 묻는 질문 (FAQ)

## 1. 회원 관련

### 1.1 회원가입
**Q: 회원가입은 어떻게 하나요?**
A: 홈페이지 우측 상단 "회원가입" 버튼을 클릭하여 가입할 수 있습니다.
가입 방법:
- 이메일 가입: 이메일 주소와 비밀번호 설정
- 소셜 로그인: 카카오, 네이버, 구글, 애플 계정으로 간편 가입
- 본인인증: 휴대폰 인증 필요 (만 14세 이상)

**Q: 만 14세 미만도 가입할 수 있나요?**
A: 만 14세 미만은 법정대리인(부모님)의 동의가 필요합니다.
부모님 인증 후 가입이 가능합니다.

**Q: 해외 거주자도 가입 가능한가요?**
A: 네, 해외 거주자도 이메일로 가입 가능합니다.
단, 일부 서비스(국내 배송 등)가 제한될 수 있습니다.

### 1.2 로그인/비밀번호
**Q: 비밀번호를 잊어버렸어요.**
A: 로그인 페이지 하단 "비밀번호 찾기"를 클릭하세요.
- 가입 이메일로 비밀번호 재설정 링크 발송
- SMS 인증으로 재설정 가능
- 소셜 가입의 경우 해당 소셜 서비스에서 비밀번호 변경

**Q: 아이디(이메일)를 잊어버렸어요.**
A: "아이디 찾기"에서 휴대폰 번호 인증으로 가입된 이메일을 확인할 수 있습니다.

**Q: 로그인이 안 돼요.**
A: 다음 사항을 확인해 주세요:
1. Caps Lock 확인
2. 소셜 로그인으로 가입했는지 확인
3. 브라우저 캐시 삭제 후 재시도
4. 여전히 문제 시 고객센터 문의

### 1.3 회원정보 변경
**Q: 이메일(아이디)을 변경할 수 있나요?**
A: 이메일 변경은 마이페이지 > 회원정보 수정에서 가능합니다.
본인인증 후 새 이메일로 변경할 수 있습니다.

**Q: 휴대폰 번호를 변경하고 싶어요.**
A: 마이페이지 > 회원정보 수정 > 휴대폰 번호 변경에서
새 번호로 인증 후 변경 가능합니다.

**Q: 배송지를 추가/변경하고 싶어요.**
A: 마이페이지 > 배송지 관리에서 최대 10개까지 배송지를 등록/관리할 수 있습니다.

### 1.4 회원 탈퇴
**Q: 회원 탈퇴는 어떻게 하나요?**
A: 마이페이지 > 회원정보 > "회원 탈퇴"를 클릭하세요.
주의사항:
- 진행 중인 주문이 있으면 탈퇴 불가
- 보유 적립금/쿠폰은 소멸됩니다
- 탈퇴 후 30일간 재가입 불가
- 관련 법률에 따라 일부 정보는 보관 (최대 5년)

**Q: 탈퇴 후 재가입 할 수 있나요?**
A: 탈퇴 후 30일이 경과하면 재가입이 가능합니다.
단, 이전 적립금, 쿠폰, 주문 이력은 복원되지 않습니다.

## 2. 결제 관련

### 2.1 결제 수단
**Q: 어떤 결제 수단을 지원하나요?**
A: 다양한 결제 수단을 지원합니다:
- 신용카드: VISA, Mastercard, 국내 전 카드사
- 체크카드: 전 카드사
- 무통장입금: 주요 은행 (입금 확인 후 1시간 이내 처리)
- 간편결제: 카카오페이, 네이버페이, 토스, 삼성페이, 애플페이
- 휴대폰 결제: SKT, KT, LGU+
- 포인트 결제: 보유 적립금 사용
- 쿠폰 결제: 보유 쿠폰 적용

**Q: 결제가 안 돼요.**
A: 다음 사항을 확인해 주세요:
1. 카드 한도 확인
2. 결제 보안 프로그램 설치 확인
3. 다른 브라우저에서 시도
4. 카드사 ARS(자동응답시스템)로 결제 차단 여부 확인
5. 여전히 문제 시 고객센터로 연락

**Q: 해외 카드로 결제할 수 있나요?**
A: VISA, Mastercard가 표시된 해외 카드로 결제 가능합니다.
일부 카드는 3D Secure 인증이 필요할 수 있습니다.

### 2.2 무통장입금
**Q: 무통장입금 계좌는 어디서 확인하나요?**
A: 주문 완료 후 주문 확인 페이지 및 이메일에서 확인 가능합니다.
마이페이지 > 주문내역에서도 확인 가능합니다.

**Q: 무통장입금 기한은 얼마인가요?**
A: 주문일로부터 **3일(72시간)** 이내 입금해야 합니다.
기한 내 미입금 시 주문이 자동 취소됩니다.

**Q: 입금자명이 다른 경우 어떻게 하나요?**
A: 입금자명이 주문자명과 다를 경우 고객센터에 연락하여
주문번호와 입금자명을 알려주시면 수동 확인해 드립니다.

### 2.3 현금영수증/세금계산서
**Q: 현금영수증은 어떻게 발급받나요?**
A: 무통장입금, 카카오페이 등 현금성 결제 시 자동 발급 신청이 가능합니다.
마이페이지 > 주문내역에서 "현금영수증 신청"을 클릭하세요.

**Q: 세금계산서 발급이 가능한가요?**
A: 사업자 회원에 한해 세금계산서 발급이 가능합니다.
주문 시 사업자번호를 입력하시면 이메일로 발송됩니다.

## 3. 적립금/포인트

### 3.1 적립
**Q: 적립금은 어떻게 쌓이나요?**
A: 다양한 방법으로 적립금을 받을 수 있습니다:
- **구매 적립**: 구매 확정 시 결제 금액의 1% 적립
- **리뷰 작성**: 텍스트 리뷰 200원, 포토 리뷰 500원
- **출석체크**: 매일 로그인 시 50원 (연속 7일 시 보너스 500원)
- **신규 가입**: 2,000원 웰컴 적립금
- **생일 축하**: 매년 생일에 3,000원 적립금

**Q: 적립금은 언제 사용할 수 있나요?**
A: 구매 확정 후 적립금이 지급되며, **1,000원 이상**부터 사용 가능합니다.

### 3.2 사용
**Q: 적립금 사용 방법은?**
A: 결제 페이지에서 "적립금 사용" 항목에 사용할 금액을 입력하세요.
- 최소 사용 금액: 1,000원
- 최대 사용 금액: 결제 금액의 최대 50%
- 적립금 + 다른 결제 수단 병행 사용 가능

**Q: 적립금 유효기간이 있나요?**
A: 네, 적립일로부터 **1년**입니다.
만료 30일 전 이메일/앱 푸시로 안내해 드립니다.

### 3.3 VIP/멤버십
**Q: 멤버십 등급은 어떻게 되나요?**
A: 최근 6개월 구매 금액 기준으로 등급이 결정됩니다:
| 등급 | 조건 (6개월 구매액) | 혜택 |
|------|-------------------|------|
| 일반 | 가입 시 기본 | 구매 적립 1% |
| 실버 | 30만원 이상 | 구매 적립 2% + 무료배송 쿠폰 2장/월 |
| 골드 | 100만원 이상 | 구매 적립 3% + 무료배송 + 5% 할인쿠폰 |
| VIP | 300만원 이상 | 구매 적립 5% + 무료배송 + 10% 할인쿠폰 + 전담 상담 |
| VVIP | 500만원 이상 | 구매 적립 7% + 전 혜택 + 신상품 우선 구매권 |

**Q: 등급은 언제 갱신되나요?**
A: 매월 1일에 최근 6개월 구매 실적 기준으로 자동 갱신됩니다.

## 4. 쿠폰

### 4.1 쿠폰 관리
**Q: 쿠폰은 어디서 확인하나요?**
A: 마이페이지 > **쿠폰함**에서 보유 중인 쿠폰을 확인할 수 있습니다.

**Q: 쿠폰은 어떻게 사용하나요?**
A: 결제 페이지에서 "쿠폰 적용" 버튼을 클릭하고 사용할 쿠폰을 선택하세요.

**Q: 쿠폰을 여러 개 사용할 수 있나요?**
A: 주문 1건당 **쿠폰 1장**만 사용 가능합니다.
적립금과 쿠폰은 동시 사용 가능합니다.

**Q: 쿠폰 유효기간이 지나면?**
A: 유효기간이 만료된 쿠폰은 자동 삭제되며 복원이 불가합니다.
만료 3일 전 알림을 보내드립니다.

### 4.2 쿠폰 종류
- **할인 쿠폰**: 정액 할인 (예: 3,000원 할인) 또는 정률 할인 (예: 10% 할인)
- **무료배송 쿠폰**: 배송비 무료
- **카테고리 쿠폰**: 특정 카테고리 상품에만 적용
- **브랜드 쿠폰**: 특정 브랜드 상품에만 적용
- **첫 구매 쿠폰**: 첫 주문 시에만 사용 가능

### 4.3 쿠폰 받는 방법
- 신규 가입 시 웰컴 쿠폰 자동 발급
- 이벤트 페이지에서 쿠폰 다운로드
- 앱 알림/이메일을 통한 특별 쿠폰
- 리뷰 작성 시 감사 쿠폰
- 생일 축하 쿠폰 (회원정보에 생일 등록 필요)

## 5. 주문 관련

### 5.1 주문 확인
**Q: 주문이 정상적으로 완료되었는지 어떻게 확인하나요?**
A: 주문 완료 시 다음과 같이 확인됩니다:
1. 주문 완료 페이지에서 주문번호 표시
2. 가입 이메일로 주문 확인 메일 발송
3. SMS/카카오톡으로 주문 안내 발송
4. 마이페이지 > 주문내역에서 확인

**Q: 주문 내역은 얼마나 보관되나요?**
A: 최근 3년간의 주문 내역을 조회할 수 있습니다.
3년 초과 이력은 고객센터에 문의해 주세요.

### 5.2 주문 취소
**Q: 주문을 취소하고 싶어요.**
A: 주문 취소는 배송 상태에 따라 다릅니다:
- **결제완료/상품준비중**: 마이페이지에서 직접 취소 가능
- **발송완료 이후**: 취소 불가 → 수령 후 반품 절차 이용
- **부분 취소**: 일부 상품만 취소 가능 (동일 주문 내)

**Q: 취소한 금액은 언제 환불되나요?**
A: 취소 즉시 환불 처리되며, 결제 수단에 따라 1~5 영업일 소요됩니다.

### 5.3 주문 변경
**Q: 주문 후 상품을 변경할 수 있나요?**
A: 주문 상품 변경은 불가합니다. 기존 주문 취소 후 재주문해 주세요.

**Q: 주문 후 수량을 변경할 수 있나요?**
A: 발송 전이라면 고객센터에 연락하여 수량 변경이 가능합니다.

## 6. 앱 관련

**Q: 앱은 어디서 다운로드하나요?**
A: iOS App Store 또는 Google Play Store에서 "쇼핑몰"을 검색하여 다운로드하세요.

**Q: 앱 알림이 안 와요.**
A: 스마트폰 설정 > 알림 > 해당 앱 > 알림 허용을 확인해 주세요.

**Q: 앱이 작동하지 않아요.**
A: 다음을 시도해 주세요:
1. 앱 강제 종료 후 재실행
2. 앱 업데이트 확인
3. 스마트폰 재부팅
4. 앱 삭제 후 재설치

## 7. 개인정보

**Q: 개인정보는 어떻게 보호되나요?**
A: 당사는 개인정보보호법을 준수하며 다음과 같이 보호합니다:
- SSL 암호화 통신
- 개인정보 암호화 저장
- 접근 권한 최소화
- 정기 보안 감사 실시
- 개인정보처리방침 공개 (홈페이지 하단)

**Q: 마케팅 수신을 거부하고 싶어요.**
A: 마이페이지 > 회원정보 > "마케팅 수신 설정"에서 변경 가능합니다.
이메일/SMS/앱 푸시 알림을 개별 설정할 수 있습니다.
