# 반품 및 교환 정책

## 1. 반품/교환 신청 기간

### 1.1 기본 규정
- **반품 기간**: 상품 수령일로부터 **7일 이내**
- **교환 기간**: 상품 수령일로부터 **7일 이내**
- **청약 철회**: 전자상거래법에 따라 수령 후 7일 이내 가능

### 1.2 기간 연장 사유
- 상품에 하자가 있는 경우: 수령 후 **30일 이내**
- 표시/광고와 다른 상품: 수령 후 **30일 이내** 또는 알게 된 날로부터 **30일**
- 배송 중 파손: 수령 후 **24시간 이내** 접수 시 즉시 처리

## 2. 반품 가능 조건

### 2.1 반품 가능한 경우
- 고객 단순 변심 (사이즈, 색상, 디자인 불만족)
- 상품 하자 (불량, 파손, 오염)
- 오배송 (주문과 다른 상품 수령)
- 표시/광고와 다른 상품
- 배송 중 파손

### 2.2 반품 불가한 경우
- 고객 책임으로 상품이 훼손/변형된 경우
- 착용/사용 흔적이 있는 경우
- 태그/라벨 제거 시
- 세탁한 경우
- 향수/화장품: 개봉 후 사용한 경우
- 식품: 개봉 후 또는 유통기한 경과
- 주문 제작 상품 (커스텀 제작)
- 디지털 콘텐츠 (다운로드 완료 후)
- 속옷/수영복: 위생상 반품 불가
- 복제 가능한 상품의 포장을 훼손한 경우

### 2.3 부분 반품
- 묶음 배송 상품 중 일부만 반품 가능
- 단, 세트 상품은 전체 반품만 가능
- 사은품이 포함된 경우 사은품도 함께 반품 필요

## 3. 반품 절차

### 3.1 온라인 반품 신청
1. **마이페이지** > **주문내역** 접속
2. 반품할 주문 선택 > **"반품 신청"** 버튼 클릭
3. 반품 사유 선택
   - 단순 변심
   - 사이즈 불일치
   - 색상 차이
   - 상품 불량/하자
   - 오배송
   - 기타 (직접 입력)
4. 상품 상태 사진 첨부 (하자/파손 시 필수)
5. 수거 방법 선택
   - 자동 수거 접수 (택배 기사 방문)
   - 직접 발송
6. 환불 방법 확인 후 **"신청 완료"**

### 3.2 고객센터 반품 신청
- 📞 전화: 1588-0000 (평일 09:00~18:00)
- 💬 카카오톡: @쇼핑몰고객센터
- 📧 이메일: return@shop.example.com

### 3.3 반품 상품 발송
- **자동 수거**: 신청 후 1~2 영업일 내 택배 기사 방문
- **직접 발송**: 아래 주소로 발송
  - 주소: 서울특별시 강남구 테헤란로 123 물류센터
  - 수신: 반품담당자
  - 반드시 주문번호 기재

### 3.4 반품 처리 과정
1. 반품 접수 → **1~2 영업일** (수거 완료)
2. 물류센터 도착 → **1 영업일** (입고 처리)
3. 상품 검수 → **1~2 영업일** (상태 확인)
4. 환불 처리 → **1~3 영업일** (결제 수단별 상이)
- **총 소요 시간: 약 3~7 영업일**

## 4. 교환 절차

### 4.1 교환 가능 범위
- **동일 상품** 사이즈/색상 변경만 가능
- 다른 상품으로의 교환은 반품 후 재주문으로 처리
- 품절 시 교환 불가 → 환불 처리

### 4.2 교환 신청 방법
1. 마이페이지 > 주문내역 > **"교환 신청"**
2. 교환 사유 및 희망 옵션 선택
3. 상품 수거 후 새 상품 발송
- **교환 소요 시간: 약 5~10 영업일**

### 4.3 교환 배송비
| 교환 사유 | 배송비 |
|-----------|--------|
| 고객 단순 변심 | 왕복 6,000원 (편도 3,000원) |
| 상품 하자 | 무료 |
| 오배송 | 무료 |
| 사이즈 교환 | 왕복 6,000원 |

## 5. 환불 안내

### 5.1 환불 처리 시점
- 반품 상품이 물류센터에 도착하여 **검수 완료 후** 환불 처리

### 5.2 결제 수단별 환불 소요 시간
| 결제 수단 | 환불 소요 시간 |
|-----------|--------------|
| 신용카드 | 3~5 영업일 (카드사별 상이) |
| 체크카드 | 3~5 영업일 |
| 무통장입금 | 1~2 영업일 |
| 카카오페이 | 즉시~1 영업일 |
| 네이버페이 | 즉시~1 영업일 |
| 토스 | 즉시~1 영업일 |
| 포인트 결제 | 즉시 복원 |
| 쿠폰 결제 | 쿠폰 재발급 (유효기간 내) |

### 5.3 환불 금액
- **전체 반품**: 상품 금액 + 배송비 (무료 배송 기준 미달 시 배송비 차감 가능)
- **부분 반품**: 반품 상품 금액만 환불 (배송비 기준은 남은 상품 기준 재계산)
- **할인/쿠폰 적용 상품**: 실 결제 금액 기준 환불
- **적립금 사용 상품**: 적립금 복원 + 현금/카드 결제분 환불

### 5.4 환불 불가 차감 항목
- 고객 변심 반품 배송비: 편도 3,000원 (왕복 6,000원)
- 상품 훼손 시: 훼손 정도에 따라 10~50% 차감 가능
- 사은품 미반납 시: 사은품 금액 차감

## 6. 반품 배송비

### 6.1 배송비 기준
| 반품 사유 | 반품 배송비 |
|-----------|-----------|
| 고객 단순 변심 | 편도 3,000원 |
| 사이즈/색상 불만족 | 편도 3,000원 |
| 상품 불량/하자 | 무료 |
| 오배송 | 무료 |
| 표시/광고 상이 | 무료 |
| 배송 중 파손 | 무료 |

### 6.2 배송비 결제 방법
- 반품 접수 시 환불 금액에서 자동 차감
- 착불 발송 시 택배비 본인 부담
- 선불 발송 후 배송비 별도 환불 (하자/오배송 시)

## 7. 특수 상품 반품

### 7.1 대형 가전/가구
- 전문 기사 방문 수거 (예약 필요)
- 수거 소요: 3~5 영업일
- 설치 상품: 설치 해체 후 수거

### 7.2 신선식품
- 수령 즉시 확인 필요
- 상태 불량 시 **24시간 이내** 사진 촬영 후 접수
- 반품 대신 재발송 또는 환불로 처리

### 7.3 해외 직구 상품
- 반품 배송비 고객 부담 (해외 반송비)
- 처리 기간: 2~4주
- 관세 환급: 세관 절차에 따라 별도 처리

## 9. 자주 묻는 반품 질문

**Q: 반품은 어떻게 신청하나요?**
A: 마이페이지 > 주문내역에서 "반품 신청" 버튼을 클릭하여 간편하게 신청할 수 있습니다.

**Q: 반품 배송비는 누가 부담하나요?**
A: 고객 단순 변심 시 편도 3,000원 고객 부담이며, 상품 하자/오배송 시 무료입니다.

**Q: 교환하려면 어떻게 해야 하나요?**
A: 마이페이지에서 교환 신청을 하시면 됩니다. 동일 상품의 사이즈/색상 변경만 가능합니다.

**Q: 환불은 얼마나 걸리나요?**
A: 반품 상품 검수 완료 후 결제 수단에 따라 1~5 영업일이 소요됩니다.

**Q: 사용한 상품도 반품할 수 있나요?**
A: 사용/착용 흔적이 있는 상품은 반품이 어렵습니다. 단, 상품 하자로 인한 경우는 예외입니다.

**Q: 세트 상품 일부만 반품할 수 있나요?**
A: 세트 상품은 전체 반품만 가능합니다. 개별 구매 상품은 부분 반품 가능합니다.

**Q: 할인 받은 상품의 환불 금액은?**
A: 실제 결제한 금액 기준으로 환불됩니다.

**Q: 사은품도 같이 반품해야 하나요?**
A: 네, 사은품도 함께 반품해야 합니다. 미반납 시 사은품 금액이 차감됩니다.

**Q: 쿠폰으로 결제한 상품 반품 시 쿠폰은?**
A: 유효기간 내라면 쿠폰이 재발급됩니다. 기간 만료 시 쿠폰 복원이 불가합니다.

**Q: 반품 검수에서 불합격되면?**
A: 검수 불합격 시 고객님께 연락드리며, 상품 반송 또는 일부 차감 후 환불로 처리됩니다.
