# 배송 안내

## 1. 국내 배송

### 1.1 배송 업체
- 기본 배송: CJ대한통운
- 대형 상품: 한진택배, 롯데택배
- 신선식품: 새벽배송 전용 (쿠팡 로켓프레시 방식)

### 1.2 배송 소요 시간
- **일반 배송**: 결제 완료 후 1~3 영업일 이내 발송
  - 수도권: 발송 후 1일
  - 지방: 발송 후 1~2일
  - 도서산간: 발송 후 2~5일 (추가 배송비 발생 가능)
- **당일 배송**: 오전 10시 이전 결제 완료 시 당일 발송 (수도권 한정)
- **새벽 배송**: 밤 10시 이전 결제 → 다음날 오전 7시 이전 도착 (서울/경기 일부)

### 1.3 배송비
- **기본 배송비**: 3,000원
- **무료 배송**: 50,000원 이상 구매 시
- **도서산간 추가 배송비**: 3,000원~5,000원
- **대형 상품 배송비**: 상품별 상이 (상품 상세페이지 참고)
- **묶음 배송**: 동일 판매자 상품에 한해 묶음 배송 가능

### 1.4 배송 불가 지역
- 군부대 (APO/FPO)
- 일부 도서산간 지역 (별도 문의 필요)

## 2. 해외 배송

### 2.1 해외 배송 가능 국가
미국, 일본, 중국, 대만, 싱가포르, 호주, 캐나다, 영국, 독일, 프랑스 등
총 30개국 배송 가능 (상세 목록은 고객센터 문의)

### 2.2 해외 배송비
| 지역 | 기본 배송비 | 추가 배송비 (kg당) |
|------|------------|-------------------|
| 아시아 (일본, 중국, 대만) | 15,000원 | 5,000원 |
| 미주 (미국, 캐나다) | 25,000원 | 8,000원 |
| 유럽 (영국, 독일, 프랑스) | 30,000원 | 10,000원 |
| 오세아니아 (호주) | 28,000원 | 9,000원 |

### 2.3 해외 배송 소요 시간
- 아시아: 5~7 영업일
- 미주: 7~14 영업일
- 유럽: 10~14 영업일
- 오세아니아: 7~10 영업일

### 2.4 해외 배송 주의사항
- 관세/부가세는 수령인 부담
- 통관 지연 시 추가 소요 가능
- 일부 상품은 해외 배송 불가 (리튬 배터리 포함 제품, 화장품 일부 등)
- 해외 배송 주문은 취소/교환/반품이 제한될 수 있음

## 3. 배송 조회

### 3.1 배송 조회 방법
1. **마이페이지 > 주문내역**: 주문 상태 및 배송 추적 가능
2. **주문 확인 이메일/SMS**: 발송 시 송장번호 자동 안내
3. **택배사 직접 조회**:
   - CJ대한통운: https://www.cjlogistics.com
   - 한진택배: https://www.hanjin.com
   - 롯데택배: https://www.lotteglogis.com

### 3.2 배송 상태 안내
- **결제완료**: 결제가 확인되었으며 상품 준비 중
- **상품준비중**: 상품을 포장하고 있습니다
- **발송완료**: 택배사에 상품이 인계되었습니다
- **배송중**: 배송 기사가 배달 중입니다
- **배송완료**: 상품이 도착했습니다

### 3.3 배송 지연 안내
- 주문 폭주 시기 (명절, 블랙프라이데이 등): 1~3일 추가 소요 가능
- 기상악화 (태풍, 폭설 등): 배송 일정 변동 가능
- 배송 지연 시 SMS/카카오톡 알림 발송

## 4. 배송지 변경

### 4.1 배송지 변경 가능 시점
- **결제완료 ~ 상품준비중**: 마이페이지에서 직접 변경 가능
- **발송완료 이후**: 변경 불가 (택배사에 직접 연락 필요)

### 4.2 변경 방법
1. 마이페이지 > 주문내역 > 해당 주문 선택
2. "배송지 변경" 버튼 클릭
3. 새 배송지 정보 입력 후 저장

## 5. 안전 배송

### 5.1 포장
- 파손 위험 상품: 에어캡 + 완충재 이중 포장
- 식품: 아이스팩 + 보냉 포장
- 고가 상품: 보험 배송 적용

### 5.2 부재 시 처리
- 경비실/무인택배함 보관 (기본 설정)
- 문 앞 배송 선택 가능
- 배송 전 전화 요청 가능 (주문 시 메모 기재)

### 5.3 분실/파손 처리
- 배송 완료 후 상품 분실: 택배사 + 쇼핑몰 공동 조사 후 보상
- 파손 상품 수령: 수령 후 24시간 이내 사진 촬영 후 고객센터 접수
- 보상 절차: 접수 → 조사(1~3일) → 재발송 또는 환불

## 6. 특수 배송

### 6.1 예약 배송
- 원하는 날짜/시간대에 수령 가능
- 추가 비용: 2,000원
- 가능 시간대: 오전(09~12시), 오후(12~18시), 저녁(18~21시)

### 6.2 선물 배송
- 선물 포장 서비스: 2,000원 추가
- 메시지 카드 동봉 가능 (무료)
- 가격표 미포함 배송 가능

### 6.3 정기 배송
- 생필품, 식품 등 정기 배송 설정 가능
- 주기: 매주/격주/매월 선택
- 정기 배송 할인: 5% 추가 할인

## 7. 자주 묻는 배송 질문

**Q: 주문 후 얼마나 걸려야 배송되나요?**
A: 일반적으로 결제 완료 후 1~3 영업일 이내에 발송되며, 발송 후 1~2일 내에 수령 가능합니다.

**Q: 배송비는 얼마인가요?**
A: 기본 배송비 3,000원이며, 50,000원 이상 구매 시 무료 배송입니다.

**Q: 배송 중 주소를 변경할 수 있나요?**
A: 발송 전까지는 마이페이지에서 변경 가능합니다. 발송 후에는 택배사에 직접 연락해 주세요.

**Q: 배송 중 상품이 파손되면 어떻게 하나요?**
A: 수령 후 24시간 이내에 파손 사진과 함께 고객센터에 접수해 주세요. 재발송 또는 환불로 처리해 드립니다.

**Q: 해외로 배송 가능한가요?**
A: 네, 30개국에 해외 배송이 가능합니다. 배송비와 소요 시간은 지역에 따라 다릅니다.

**Q: 당일 배송이 가능한가요?**
A: 오전 10시 이전 결제 완료 시 수도권 한정으로 당일 배송이 가능합니다.

**Q: 택배 기사에게 직접 전화 요청할 수 있나요?**
A: 주문 시 배송 메모에 "배송 전 전화 요청"을 기재해 주시면 됩니다.

**Q: 부재 시 택배는 어디에 보관되나요?**
A: 기본적으로 경비실 또는 무인택배함에 보관됩니다. 문 앞 배송을 원하시면 배송 메모에 기재해 주세요.
